🧩JSX 语法与组件基础

JSX是React的语法扩展，允许在JavaScript中编写类似HTML的代码。它使组件的结构更加直观和易于理解。

JSX核心概念：
  • 语法糖：JSX会被编译为React.createElement()调用
  • 表达式：使用{}包裹JavaScript表达式
  • 属性：使用camelCase命名，如className、onClick
  • 组件：自定义组件必须以大写字母开头

🔄State 状态管理与 Hooks

State是React组件的核心概念，用于管理组件的内部状态。Hooks是React 16.8引入的新特性，让函数组件也能使用状态和其他React特性。

常用React Hooks：
  • useState：管理组件状态
  • useEffect：处理副作用（生命周期）
  • useContext：使用Context API
  • useReducer：复杂状态管理
  • useMemo：性能优化，缓存计算结果
  • useCallback：缓存函数引用
// useState Hook 示例 import React, { useState } from 'react'; function Counter() { // 声明状态变量 const [count, setCount] = useState(0); const [name, setName] = useState(''); // 事件处理函数 const increment = () => setCount(count + 1); const decrement = () => setCount(count - 1); const reset = () => setCount(0); return ( <div className="counter"> <h3>计数器: {count}</h3> <button onClick={increment}>+1</button> <button onClick={decrement}>-1</button> <button onClick={reset}>重置</button> <div> <input type="text" value={name} onChange={(e) => setName(e.target.value)} placeholder="输入你的名字" /> <p>你好，{name || '访客'}！</p> </div> </div> ); } // useEffect Hook 示例 import React, { useState, useEffect } from 'react'; function Timer() { const [seconds, setSeconds] = useState(0); const [isRunning, setIsRunning] = useState(false); // 组件挂载和更新时执行 useEffect(() => { let interval = null; if (isRunning) { interval = setInterval(() => { setSeconds(seconds => seconds + 1); }, 1000); } else if (!isRunning && seconds !== 0) { clearInterval(interval); } // 清理函数 return () => clearInterval(interval); }, [isRunning, seconds]); return ( <div className="timer"> <h3>计时器: {seconds}秒</h3> <button onClick={() => setIsRunning(!isRunning)}> {isRunning ? '暂停' : '开始'} </button> <button onClick={() => setSeconds(0)}>重置</button> </div> ); } // 自定义Hook示例 function useLocalStorage(key, initialValue) { const [storedValue, setStoredValue] = useState(() => { try { const item = window.localStorage.getItem(key); return item ? JSON.parse(item) : initialValue; } catch (error) { console.log(error); return initialValue; } }); const setValue = (value) => { try { setStoredValue(value); window.localStorage.setItem(key, JSON.stringify(value)); } catch (error) { console.log(error); } }; return [storedValue, setValue]; } // 使用自定义Hook function Settings() { const [theme, setTheme] = useLocalStorage('theme', 'light'); const [language, setLanguage] = useLocalStorage('language', 'zh'); return ( <div> <h3>设置</h3> <label> 主题: <select value={theme} onChange={(e) => setTheme(e.target.value)}> <option value="light">浅色</option> <option value="dark">深色</option> </select> </label> </div> ); }

🎯事件处理与表单

React中的事件处理采用合成事件系统，提供了跨浏览器的一致性体验。表单处理是Web应用的重要组成部分。

React事件处理特点：
  • 合成事件：SyntheticEvent，跨浏览器兼容
  • 事件委托：所有事件都绑定到根元素
  • 阻止默认：必须显式调用preventDefault()
  • this绑定：需要注意函数中this的指向
// 事件处理示例 function EventDemo() { const [message, setMessage] = useState(''); // 点击事件 const handleClick = (event) => { console.log('按钮被点击', event.type); setMessage('按钮被点击了！'); }; // 鼠标事件 const handleMouseEnter = () => { setMessage('鼠标进入了！'); }; // 键盘事件 const handleKeyPress = (event) => { if (event.key === 'Enter') { setMessage(`你按下了回车键！`); } }; return ( <div> <button onClick={handleClick} onMouseEnter={handleMouseEnter} > 点击或悬停 </button> <input type="text" onKeyPress={handleKeyPress} placeholder="按回车键试试" /> <p>{message}</p> </div> ); } // 表单处理示例 function ContactForm() { const [formData, setFormData] = useState({ name: '', email: '', message: '', category: 'general', subscribe: false }); // 处理输入变化 const handleInputChange = (event) => { const { name, value, type, checked } = event.target; setFormData(prevState => ({ ...prevState, [name]: type === 'checkbox' ? checked : value })); }; // 表单提交 const handleSubmit = (event) => { event.preventDefault(); console.log('表单数据:', formData); // 表单验证 if (!formData.name || !formData.email) { alert('请填写必填字段！'); return; } // 提交逻辑 alert('表单提交成功！'); }; return ( <form onSubmit={handleSubmit}> <div> <label> 姓名 *: <input type="text" name="name" value={formData.name} onChange={handleInputChange} required /> </label> </div> <div> <label> 邮箱 *: <input type="email" name="email" value={formData.email} onChange={handleInputChange} required /> </label> </div> <div> <label> 类别: <select name="category" value={formData.category} onChange={handleInputChange}> <option value="general">一般咨询</option> <option value="support">技术支持</option> <option value="feedback">意见反馈</option> </select> </label> </div> <div> <label> 消息: <textarea name="message" value={formData.message} onChange={handleInputChange} rows="4" /> </label> </div> <div> <label> <input type="checkbox" name="subscribe" checked={formData.subscribe} onChange={handleInputChange} /> 订阅邮件通知 </label> </div> <button type="submit">提交</button> </form> ); } // Todo应用示例 function TodoApp() { const [todos, setTodos] = useState([]); const [inputValue, setInputValue] = useState(''); // 添加待办事项 const addTodo = () => { if (inputValue.trim()) { setTodos([...todos, { id: Date.now(), text: inputValue, completed: false }]); setInputValue(''); } }; // 切换完成状态 const toggleTodo = (id) => { setTodos(todos.map(todo => todo.id === id ? { ...todo, completed: !todo.completed } : todo )); }; // 删除待办事项 const deleteTodo = (id) => { setTodos(todos.filter(todo => todo.id !== id)); }; return ( <div className="todo-app"> <h3>待办事项</h3> <div> <input type="text" value={inputValue} onChange={(e) => setInputValue(e.target.value)} onKeyPress={(e) => e.key === 'Enter' && addTodo()} placeholder="添加新的待办事项" /> <button onClick={addTodo}>添加</button> </div> <ul> {todos.map(todo => ( <li key={todo.id} className={todo.completed ? 'completed' : ''}> <span onClick={() => toggleTodo(todo.id)}> {todo.text} </span> <button onClick={() => deleteTodo(todo.id)}>删除</button> </li> ))} </ul> </div> ); }

🔄 React 生命周期

React组件有完整的生命周期，从创建到销毁经历多个阶段。理解生命周期有助于优化性能和处理副作用。

函数组件生命周期（Hooks）

挂载 → 更新 → 卸载

useEffect 可以模拟所有生命周期方法

🚀 挂载阶段

  • 组件创建：初始化state和props
  • 首次渲染：执行render方法
  • DOM更新：组件挂载到DOM
  • useEffect：执行副作用

🔄 更新阶段

  • props变化：父组件传入新props
  • state变化：调用setState更新状态
  • 重新渲染：执行render方法
  • DOM更新：更新变化的部分

💥 卸载阶段

  • 组件移除：从DOM中移除组件
  • 清理工作：清除定时器、事件监听
  • useEffect清理：执行清理函数
  • 内存释放：释放组件占用的内存

⚡ 性能优化

  • React.memo：避免不必要的重渲染
  • useMemo：缓存计算结果
  • useCallback：缓存函数引用
  • 懒加载：React.lazy和Suspense
