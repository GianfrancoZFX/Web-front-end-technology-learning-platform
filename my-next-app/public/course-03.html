第三周 - CSS基础知识练习

学习目标：掌握CSS的基础语法、选择器和三种样式表的使用方式

CSS样式表的三种使用方式

  • 外部样式表 - 通过<link>标签引入外部CSS文件
  • 内部样式表 - 在<style>标签中定义样式
  • 内联样式表 - 直接在HTML元素的style属性中定义

CSS基础语法示例

/* CSS语法结构 */ 选择器 { 属性名: 属性值; 属性名: 属性值; } /* 示例 */ h1 { color: blue; font-size: 24px; text-align: center; }

CSS选择器类型

/* 元素选择器 */ p { color: black; } /* 类选择器 */ .highlight { background-color: yellow; } /* ID选择器 */ #header { font-size: 24px; } /* 后代选择器 */ div p { margin: 10px; }
